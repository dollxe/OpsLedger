# 대기배출시설 운영기록부 관리

계량기 수치를 날짜별로 기록하고, 운영일지 Excel 파일을 자동으로 만들어주는 프로그램입니다.

---

## 시작하기

### 웹 버전 (설치 없음)

아래 링크에서 바로 사용할 수 있습니다.

**👉 [앱 열기](https://your-app.streamlit.app)** ← 배포 후 실제 URL로 교체

- 회원가입 → 이메일 인증 → 로그인
- 내 계정으로 로그인하면 **내 데이터만** 보입니다 (다른 사람과 섞이지 않음)

---

### PC 설치 버전 (오프라인 사용)

별도로 전달받은 압축 파일을 풀고 아래 순서대로 진행하세요.

**처음 한 번만:**
1. `바탕화면에_추가.bat` 더블클릭 → 바탕화면에 아이콘 생성
2. 바탕화면 아이콘 더블클릭
3. 자동으로 설치가 진행됩니다 (인터넷 연결 필요, 약 5~10분 소요)
4. 설치 완료 후 브라우저가 자동으로 열립니다

**이후 매번:**
- 바탕화면 아이콘 더블클릭 → 브라우저 자동 오픈

> ※ PC 버전은 데이터가 내 컴퓨터에만 저장됩니다.

---

## 사용 방법

### 📝 데이터 입력
1. 날짜를 선택합니다
2. 동주·신성 계량기 수치를 입력합니다
3. **확인 및 저장** 버튼을 누릅니다

> 며칠치를 한번에 올릴 경우, 빠진 날짜는 자동으로 채워집니다.
> 토·일·공휴일은 자동으로 제외됩니다.

### 📊 데이터 조회
- 날짜 범위를 설정하면 해당 기간 데이터를 표로 확인할 수 있습니다
- 노란색 행은 자동으로 채워진(보간) 데이터입니다
- 특정 날짜 행을 선택해 삭제할 수 있습니다

### 📤 Excel 내보내기
1. 내보낼 날짜 범위를 선택합니다
2. **동주** 또는 **신성** 버튼을 눌러 Excel 파일을 생성합니다
3. 다운로드 버튼으로 저장합니다

### ⚙️ 설정 (웹 버전)
- **전체 데이터 삭제**: `전체삭제` 입력 후 버튼 클릭
- **실수로 삭제했다면**: 12시간 이내에 **삭제 취소(복원)** 버튼으로 되돌릴 수 있습니다
- **로그아웃**: 설정 탭 하단
